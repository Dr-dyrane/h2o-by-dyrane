## Experience Design // The Alexander Canon

H₂O implements the **Alexander UI/UX Canon (v1.0)**, a set of ruthless design laws favoring depth over paint and presence over decoration.

### 1. The Soap Bubble Aesthetic
We have moved beyond solid glass approximations to a zero-paint **Soap Bubble** material. 
*   **Zero-Paint Transparency**: 100% optical clarity. The interface is a clear membrane, not a plastic sheet.
*   **Thin-Film Interference**: Iridescent rim highlights (emerald, violet, indigo) simulate the shimmering specular highlights of a liquid bubble.
*   **Vibrancy Stack**: High-index `saturate` and `brightness` filters pull color from the background *into* the material membrane, creating a refractive "glued" effect.

### 2. Global Border Purge
To achieve ultimate "Premium Lightness," we have purged all physical strokes from the application.
*   **Stroke-Free UI**: 0px borders globally. Structure is defined exclusively by background density, shadow depth, and refraction.
*   **Spatial Memory**: Transitions and depth-based layering (squircles) provide orientation without the visual debt of borders.
